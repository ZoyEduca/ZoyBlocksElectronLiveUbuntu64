Blockly Games : Sobre

Blockly Games é uma série de jogos educacionais que ensinam programação. Ele é projetado para crianças que não tiveram experiência prévia com programação de computadores. Ao final desses jogos, os jogadores estarão prontos para usar linguagens convencionais baseadas em texto.

Puzzle é uma rápida introdução às formas do Blockly e como as peças se conectam.

Maze é uma introdução a laços (loops) e condicionais. Ele começa de forma simples, mas cada nível é mais desafiador que o anterior.

Bird é um mergulho profundo em condicionais. O fluxo de controle é explorado com condições cada vez mais complexas.

Turtle é um mergulho profundo em laços. Use laços aninhados para pintar um desenho. Depois publique sua arte para que o mundo possa ver.

Movie é uma introdução a equações matemáticas. Use matemática para animar um filme. Depois publique seu filme para que o mundo possa assistir.

Music é uma introdução a funções. Use funções para compor música. Depois publique sua música para que o mundo possa ouvir.

Pond Tutor apresenta programação baseada em texto. Os níveis alternam entre blocos e JavaScript real em um editor de texto.

Pond é uma competição aberta para programar o pato mais inteligente. Use blocos ou JavaScript.

Blockly Games é um projeto do Google para incentivar os programadores de amanhã. Os jogos são projetados para serem autodidatas e no seu próprio ritmo. Sinta-se à vontade para usar o Blockly Games tanto dentro quanto fora da sala de aula. Todo o código é open-source; veja o site do desenvolvedor para mais informações ou para baixar arquivos zip para uso offline.

Desenvolvedores, professores e pais são bem-vindos para enviar feedback enquanto continuamos a crescer.
